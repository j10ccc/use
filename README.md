# Use

Things I am using.

## Editor

- [VSCodium](https://vscodium.com) (For usual development)
	- With [vim extension](https://open-vsx.org/extension/vscodevim/vim)
    - View config in [j10ccc/nixos-config/vscodium](https://github.com/j10ccc/nixos-config/blob/master/modules/vscodium/settings.json)
- [Neovim](https://neovim.io)
	- With core plugin [AstroNvim](https://astronvim.com)
	- View config in [j10ccc/astronvim-config](https://github.com/j10ccc/astronvim-config)
- Theme: [Nord](https://www.nordtheme.com)
- Font: Monaspace Neon(cv11)
- Formatting & Linting: ESLint with [`@zjutjh/eslint-config`](https://github.com/zjutjh/eslint-config)

## CLI

- [`wezterm`](https://wezfurlong.org/wezterm/index.html) - Terminal Emulator
	- View config in [j10ccc/nixos-config/wezterm](https://github.com/j10ccc/nixos-config/tree/master/modules/wezterm)
- [`fish`](https://github.com/fish-shell/fish-shell)  - Shell
	-  View config in [j10ccc/nixos-config/fish](https://github.com/j10ccc/nixos-config/tree/master/modules/fish)
- [`pnpm`](https://pnpm.io/) - Preferred package managers, also for monorepo
- [`zoxide`](https://github.com/ajeetdsouza/zoxide) - `cd` alternative
- [`bun`](https://github.com/oven-sh/bun) - Run TypeScript files without compilation
- [`ripgrep`](https://github.com/BurntSushi/ripgrep) - Match content just start with `rg`!
- [`fd`](https://github.com/sharkdp/fd) - Simple, fast and user-friendly alternative to `find`
- [`eslint`](https://eslint.org/) - Linter & formatter
	- View config in [zjutjh/eslint-config](https://github.com/zjutjh/eslint-config)
- [`antfu/ni`](https://github.com/antfu-collective/ni) - NPM commands shorthands
- [`eza`](https://github.com/eza-community/eza) - Modern replacement for `ls`
- [`bat`](https://github.com/sharkdp/bat) - Alternative to `cat`, better with `prettybat`
- [`direnv`](https://direnv.net/) - Project-specific environment variables loader
- [`devbox`](https://www.jetpack.io/devbox/) - Predictable development environments based on Nix
- [`whistle`](https://github.com/avwo/whistle) - Development HTTP proxy
- [`gemini-cli`](https://github.com/google-gemini/gemini-cli) - ONE for all coding Agent
